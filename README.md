# Shop2
# 🛒 Shop Management System

Hệ thống quản lý cửa hàng được xây dựng bằng **Spring Boot** và **MySQL**.  
Ứng dụng hỗ trợ quản lý người dùng, sản phẩm và đơn hàng với phân quyền rõ ràng giữa ADMIN và USER.

---

## 📖 Description

Shop Management System là một ứng dụng web giúp quản lý hoạt động bán hàng cơ bản:

- Quản lý sản phẩm
- Quản lý người dùng
- Quản lý đơn hàng
- Phân quyền truy cập (ADMIN / USER)

Dự án được xây dựng theo mô hình MVC và sử dụng Spring Data JPA để thao tác với database.

---

## 🚀 Features

### 👤 Authentication & Authorization
- Đăng ký tài khoản
- Đăng nhập
- Phân quyền ADMIN / USER
- Bảo mật bằng Spring Security

### 📦 Product Management (ADMIN)
- Thêm sản phẩm
- Sửa sản phẩm
- Xóa sản phẩm
- Xem danh sách sản phẩm

### 🛍 Order Management
- Thêm sản phẩm vào giỏ hàng
- Tạo đơn hàng
- Xem lịch sử đơn hàng
